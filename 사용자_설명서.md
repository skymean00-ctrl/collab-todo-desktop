## 1. 설치하기

### 설치 파일 받기

관리자에게 아래 링크 또는 파일을 요청하세요.

```
https://api.skymeanai.com/updates/CollabTodo Setup 1.0.5.exe
```

> 직접 다운로드가 어렵다면 관리자가 USB 또는 메신저로 설치 파일을 전달해드립니다.

---

### 설치 진행

**1단계** — 설치 파일(`CollabTodo Setup 1.0.5.exe`)을 더블클릭합니다.

**2단계** — Windows 보안 경고가 뜨면 **"추가 정보"** 클릭 → **"실행"** 클릭합니다.

```
Windows Defender SmartScreen
이 앱을 시작하지 않도록 PC를 보호했습니다.

[추가 정보] ← 클릭
  ↓
[실행] ← 클릭
```

> 앱이 개인 서명 인증서 없이 배포되어 이 경고가 표시됩니다. 안전한 파일입니다.

**3단계** — 설치 경로를 선택하고 **"설치"** 버튼을 클릭합니다.

- 기본 경로: `C:\Users\사용자명\AppData\Local\Programs\CollabTodo`
- 변경이 필요하면 **"찾아보기"** 버튼으로 경로 지정

**4단계** — 설치가 완료되면 **"CollabTodo 실행"** 체크박스를 선택하고 **"마침"** 클릭합니다.

## 2. 최초 실행 - 서버 주소 설정

앱을 **처음 실행하면** 서버 주소 설정 화면이 나타납니다.
이 단계는 **최초 1회만** 설정하면 됩니다.

### 화면 설명

```
┌─────────────────────────────────┐
│        CollabTodo 설정          │
│    처음 한 번만 설정하면 됩니다    │
│                                 │
│  서버 주소                       │
│  [http://___________________]   │
│                                 │
│  [ 연결 테스트 ]  [ 저장 후 시작 ] │
└─────────────────────────────────┘
```

### 입력 방법

**서버 주소**란에 관리자에게 받은 주소를 입력합니다.

| 환경 | 주소 예시 |
|------|---------|
| 공개 서버 (외부 접속) | `https://api.skymeanai.com` |
| 사내망 | `http://192.168.1.100:8000` |
| 로컬 테스트 | `http://localhost:8000` |

### 연결 확인

1. 주소 입력 후 **"연결 테스트"** 버튼 클릭
2. **"서버 연결 성공"** (초록색) 메시지 확인
3. **"저장 후 시작"** 버튼 클릭 → 로그인 화면으로 이동

> 연결 실패 시: 주소가 정확한지 확인하거나 관리자에게 문의하세요.

## 3. 회원가입

로그인 화면 하단의 **"회원가입"** 링크를 클릭합니다.

### 입력 항목

| 항목 | 설명 | 예시 |
|------|------|------|
| **이름** | 실제 이름 | 홍길동 |
| **회사 이메일** | 로그인 아이디로 사용됨 | hong@company.com |
| **부서** | 소속 부서 선택 (관리자가 등록한 부서 목록) | 개발팀 |
| **비밀번호** | 아래 조건 참고 | |
| **비밀번호 확인** | 위와 동일하게 입력 | |

### 비밀번호 조건

비밀번호는 아래 **4가지를 모두** 포함해야 합니다:

- 최소 **8자 이상**
- **대문자** 포함 (A-Z)
- **소문자** 포함 (a-z)
- **숫자** 포함 (0-9)
- **특수문자** 포함 (`!@#$%^&*` 등)

**예시** (사용 가능): `MyPass123!`
**예시** (사용 불가): `password123` (대문자·특수문자 없음)

### 가입 완료

**"가입하기"** 버튼 클릭 → 자동 로그인 후 대시보드로 이동합니다.

> 부서 목록이 표시되지 않는 경우, 직접 부서명을 입력하면 됩니다.

> 관리자에게 이메일 또는 메신저로 문의해주세요.

## 6. 앱 업데이트

새 버전이 출시되면 **자동으로 알림**이 옵니다.

### 업데이트 방법

**방법 1 — 알림 팝업**
- 실행 중 "새 버전이 있습니다" 팝업 → **"다운로드"** 클릭 → 설치

**방법 2 — 트레이 아이콘**
1. 오른쪽 하단 작업표시줄의 CollabTodo 아이콘 **우클릭**
2. **"업데이트 확인"** 클릭
3. 새 버전 발견 시 → **"다운로드"** → 완료 후 **"지금 설치"** 클릭

```
트레이 우클릭 메뉴:
┌──────────────────────────┐
│ CollabTodo v1.0.5 열기   │
│ 업데이트 확인             │  ← 클릭
│ 종료                     │
└──────────────────────────┘
```

## 7. 자주 묻는 질문 (FAQ)

**Q. 설치 후 화면이 안 나와요.**
A. 작업표시줄 우측 하단 트레이 영역에서 CollabTodo 아이콘을 더블클릭해 주세요.

---

**Q. "서버에 연결할 수 없습니다"가 뜨면 어떻게 하나요?**
A. 다음을 확인하세요:
- 서버 주소가 정확한지 (`https://` 또는 `http://` 포함 여부)
- 인터넷 연결 상태
- 사내망 사용 중이라면 VPN 또는 사무실 네트워크 연결 여부
- 그래도 안 되면 관리자에게 문의

---

**Q. 부서 선택 목록이 비어있어요.**
A. 관리자가 아직 부서를 등록하지 않은 경우입니다. 직접 부서명을 텍스트로 입력하거나 관리자에게 등록을 요청하세요.

---

**Q. "이미 사용 중인 이메일입니다" 오류가 뜨면?**
A. 해당 이메일로 이미 계정이 만들어진 상태입니다. 로그인 화면으로 돌아가 로그인하거나, 비밀번호 초기화를 관리자에게 요청하세요.

---

**Q. 비밀번호 조건을 맞췄는데도 가입이 안 돼요.**
A. 아래를 확인하세요:
- 비밀번호 확인란이 정확히 일치하는지
- 대문자·소문자·숫자·특수문자 4가지 모두 포함됐는지
- 8자 이상인지

---

**Q. 앱을 삭제하고 다시 설치하면 서버 주소를 다시 설정해야 하나요?**
A. 네, 앱을 완전히 삭제하면 설정이 초기화됩니다. 재설치 후 서버 주소를 다시 입력해주세요.

---

> **문의**: 관리자에게 연락하거나 사내 메신저로 문의해주세요.
