# 배포 가이드 - Collab Todo Desktop

## 개요

이 문서는 Collab Todo Desktop을 불특정 다수의 사용자에게 배포하는 방법을 설명합니다.

## 설치 파일 특징

### ✅ 완전 독립 실행형

빌드된 설치 파일(`CollabTodoDesktop-Setup-1.0.0.exe`)은 다음을 모두 포함합니다:

1. **Python 런타임**
   - 사용자가 Python을 별도로 설치할 필요 없음
   - Python 3.11+ 런타임이 포함됨

2. **모든 Python 라이브러리**
   - PyQt5 (GUI 프레임워크)
   - mysql-connector-python (데이터베이스 연결)
   - requests (HTTP 클라이언트)
   - 기타 모든 의존성

3. **애플리케이션 실행 파일**
   - CollabToDoDesktop.exe
   - 모든 리소스 포함

4. **설치 구성 요소**
   - 시작 메뉴 바로가기
   - 바탕화면 바로가기 (선택적)
   - 제거 프로그램

### 사용자 요구사항

사용자는 다음만 필요합니다:

- ✅ Windows 10 이상 (Windows 7 이상도 가능하나 권장하지 않음)
- ✅ 설치 파일 실행 권한
- ✅ 약 200MB의 디스크 공간

**불필요한 것들:**
- ❌ Python 설치 불필요
- ❌ 추가 라이브러리 설치 불필요
- ❌ 관리자 권한 불필요 (기본 설정)

## 빌드 방법

### Windows에서 빌드

```batch
# 1. 프로젝트 디렉토리로 이동
cd collab-todo-desktop

# 2. 완전 자동 빌드 스크립트 실행
build_installer_complete.bat
```

빌드가 완료되면:
```
installer\CollabTodoDesktop-Setup-1.0.0.exe
```

### 빌드 전 확인사항

1. **Python 3.11+** 설치 확인
2. **Inno Setup 6** 설치 확인
3. **인터넷 연결** (의존성 다운로드용)

## 배포 방법

### 1. 직접 배포

빌드된 설치 파일을 다음 방법으로 배포:

- **이메일 첨부**
- **파일 공유 서비스** (Google Drive, Dropbox 등)
- **내부 네트워크 공유 폴더**
- **USB 드라이브**

### 2. 웹 배포

웹 서버에 업로드하여 다운로드 링크 제공:

```
https://your-domain.com/downloads/CollabTodoDesktop-Setup-1.0.0.exe
```

### 3. 자동 업데이트 (향후 구현)

향후 버전에서 자동 업데이트 기능 추가 예정.

## 사용자 설치 가이드

### 간단한 설치 절차

1. **설치 파일 다운로드**
   - `CollabTodoDesktop-Setup-1.0.0.exe` 파일 다운로드

2. **설치 파일 실행**
   - 다운로드한 파일을 더블클릭
   - 사용자 계정 컨트롤(UAC) 프롬프트에서 "예" 클릭

3. **설치 마법사 진행**
   - 설치 위치 선택 (기본값 사용 권장)
   - 바탕화면 바로가기 선택 (선택적)
   - "설치" 클릭

4. **설치 완료**
   - 설치가 완료되면 "Collab Todo Desktop 실행" 옵션 선택 가능
   - 또는 시작 메뉴에서 프로그램 실행

### 초기 설정

프로그램을 처음 실행하면:

1. **데이터베이스 연결 설정**
   - 환경 변수 설정 필요
   - 자세한 내용: `docs\INSTALLATION_GUIDE.md`

2. **사용자 선택**
   - 활성 사용자 목록에서 사용자 선택
   - 선택한 사용자의 작업이 표시됨

## 문제 해결

### 설치 파일이 실행되지 않습니다

**가능한 원인:**
- 바이러스 백신 소프트웨어 차단
- Windows Defender 스마트 스크린 경고

**해결 방법:**
1. 바이러스 백신 소프트웨어에서 예외 추가
2. Windows Defender에서 "추가 정보" → "실행" 클릭

### 프로그램이 시작되지 않습니다

**가능한 원인:**
- 데이터베이스 연결 설정 누락
- 방화벽 차단

**해결 방법:**
1. 환경 변수 설정 확인
2. 방화벽에서 MySQL 포트(3306) 허용
3. 네트워크 연결 확인

### 자세한 문제 해결

`docs\INSTALLATION_GUIDE.md`의 "문제 해결" 섹션 참조.

## 버전 관리

### 파일 명명 규칙

```
CollabTodoDesktop-Setup-{버전}.exe
예: CollabTodoDesktop-Setup-1.0.0.exe
```

### 버전 업데이트 시

1. `installer.iss`에서 `AppVersion` 수정
2. `pyinstaller.spec`에서 버전 정보 업데이트 (선택적)
3. 빌드 스크립트 재실행
4. 새 설치 파일 배포

## 보안 고려사항

### 코드 서명 (선택적)

프로덕션 배포 시 코드 서명 인증서를 사용하여 설치 파일에 서명하는 것을 권장합니다:

1. 코드 서명 인증서 구매
2. Inno Setup 스크립트에 서명 정보 추가
3. 서명된 설치 파일 배포

이렇게 하면 Windows Defender 경고를 줄일 수 있습니다.

### 바이러스 백신 대응

일부 바이러스 백신 소프트웨어가 PyInstaller로 빌드된 파일을 의심스러운 것으로 표시할 수 있습니다.

**대응 방법:**
- 코드 서명 사용
- 주요 바이러스 백신 업체에 화이트리스트 신청
- 사용자에게 사전 안내
